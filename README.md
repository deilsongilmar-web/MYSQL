# Projeto de Banco de Dados MySQL 🗄️

Este repositório contém uma coleção de scripts SQL, modelagens de dados e exercícios práticos desenvolvidos para consolidar conhecimentos em bancos de dados relacionais utilizando o **MySQL**.

## 🚀 Sobre o Projeto

O objetivo deste projeto é documentar meu aprendizado e evolução em SQL, cobrindo desde consultas básicas até modelagens complexas e integração com back-end (Node.js/Prisma).

### 🛠️ Tecnologias e Ferramentas
* **Banco de Dados:** MySQL
* **Modelagem:** MySQL Workbench / DBDesigner
* **Ambiente:** Docker (opcional) / XAMPP

---

## 📂 Estrutura do Repositório

* `/scripts`: Comandos DDL (Data Definition Language) e DML (Data Manipulation Language).
* `/models`: Diagramas Entidade-Relacionamento (DER).
* `/desafios`: Resolução de problemas práticos e queries complexas.

---

## 📑 Conceitos Aplicados

Durante o desenvolvimento deste repositório, explorei os seguintes tópicos:
- [x] Criação de Bancos de Dados e Tabelas (`CREATE`, `ALTER`, `DROP`).
- [x] Manipulação de dados (`INSERT`, `UPDATE`, `DELETE`).
- [x] Consultas avançadas (`SELECT` com `JOINs`, `GROUP BY`, `HAVING`).
- [x] Criação de Views, Triggers e Stored Procedures.
- [x] Normalização de Dados (1ª, 2ª e 3ª Formas Normais).
- [x] Integração de banco de dados com APIs Node.js e Prisma ORM.
